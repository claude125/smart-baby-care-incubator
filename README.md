# Smart Baby Care Incubator

IoT-based neonatal incubator designed to improve neonatal care in resource-limited healthcare environments.

## Overview

The Smart Baby Care Incubator integrates IoT sensors, artificial intelligence, and a hybrid power system to monitor and support premature and vulnerable newborns.

The system continuously monitors environmental conditions and infant vital signs while providing automated alerts and predictive health insights.

## Key Features

- Temperature and humidity regulation
- Oxygen flow monitoring
- Infant body temperature monitoring
- Heart rate and respiratory monitoring
- SpO₂ (oxygen saturation) monitoring
- Weight monitoring and growth tracking
- Cry detection and sound analysis
- Automated alerts for abnormal conditions
- SMS and email notifications for caregivers
- Live video monitoring
- AI-based health risk prediction

## Technologies

- IoT sensors
- Embedded systems
- Python
- AI / Machine Learning
- Cloud monitoring
- SMS notification systems

## System Architecture

Sensors → Microcontroller → Data Processing → AI Analysis → Alerts & Monitoring Dashboard

## Applications

- Neonatal intensive care units
- Rural healthcare facilities
- Low-resource hospitals
- Remote health monitoring
