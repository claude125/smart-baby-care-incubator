## Research Publication

This project is based on the research work:

**Smart Baby Care Incubator with IoT Monitoring**

Published in the *International Journal of Internet of Things and Cyber-Assurance (IJITCA)*.

The research focuses on integrating IoT sensors, artificial intelligence, and hybrid power systems to improve neonatal care in resource-limited healthcare environments.

## System Architecture

The system integrates multiple components:

Sensors → Microcontroller → Data Processing → AI Analysis → Monitoring Dashboard → Alert System

Main components include:

- Environmental sensors (temperature, humidity, air quality)
- Infant vital sign monitoring sensors
- AI health risk analysis module
- Notification system (SMS / Email alerts)
- Remote monitoring dashboard
